# AGENTS.md examples

`AGENTS.md` is the canonical instruction source for coding agents in a project.
These examples show useful density for different project shapes. Copy and adapt
the closest example; do not reuse one blindly.

Keep a project-specific `AGENTS.md` short enough to read before work starts. It
should define:

- goal
- success criteria
- guardrails
- reuse-first policy
- validation
- handoff
- references

Use these examples as documentation only. The scaffold helpers do not copy them
into generated projects by default.
