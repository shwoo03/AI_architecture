# Claude Agent SDK app example

Use this shape for programmatic Claude Code-like workflows. For many projects,
normal Claude Code plus a generated `CLAUDE.md` adapter is enough.

Use Claude Agent SDK only when programmatic control is needed for options,
permissions, sessions, hooks, or optional subagents.

Official links live in `templates/links.md`.

Suggested shape:

```text
AGENTS.md
docs/
  PROJECT_PROFILE.md
  SECURITY.md
  REFERENCES.md
claude-agent/
  README.md
  options-policy.md
  permission-policy.md
  session-policy.md
  hooks-policy.md
  subagent-policy.md
```
