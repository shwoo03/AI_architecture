# Scope

Check documentation consistency across:

- canonical templates
- recipes
- profiles
- generated adapter notes
- examples

# Rules

- Keep official links centralized.
- Do not add runtime behavior.
- Do not make generated adapters the canonical source.
- Flag drift between `AGENTS.md` and harness adapter guidance.
