# Eval feedback examples

Use these examples for agent-runtime apps and repeated agent failures.

These examples are not copied into scaffolded projects by default. Use with
`recipes/eval-feedback-loop.md`.

## Files

- `eval-plan.md`: lightweight eval plan template.
- `trace-review.md`: trace review template.
- `regression-cases.md`: regression case template.
- `filled-trace-review-reuse-first-failure.md`: filled example showing a reuse-first failure becoming a regression case.
- `release-gate.md`: documentation-only release gate example.
