# Research material examples

This folder shows optional research-heavy project artifacts.

These examples are not copied into scaffolded projects by default. Use them only
when papers, blogs, official docs, issues, or repos materially affect decisions.

See `recipes/research-material-management.md`.

## Files

- `source-card.md`: single-source note template.
- `research-brief.md`: question-oriented research brief.
- `synthesis.md`: cross-source synthesis.
- `applied-change-record.md`: record of a research-driven change.
- `filled-research-capture-official-sdk-choice.md`: compact filled example.
