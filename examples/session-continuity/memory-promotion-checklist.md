# Memory promotion checklist

## Keep in handoff

- one-session todo
- temporary blocker
- current branch status
- validation output summary

## Move to AGENTS.md

- repeated agent behavior rule
- validation rule every agent must follow
- destructive operation rule

## Move to PROJECT_PROFILE.md

- project scope
- non-goals
- environment assumptions
- success criteria

## Move to SECURITY.md

- secret location policy
- tool permission policy
- deployment/migration approval rule

## Move to REFERENCES.md

- package adoption
- rejected alternatives
- license/provenance
- copied source record

## Move to research/

- detailed paper analysis
- blog comparison
- multi-source synthesis
- source excerpts or notes
