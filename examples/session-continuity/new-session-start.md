# New session start checklist

1. Read `AGENTS.md`.
2. Read `docs/PROJECT_PROFILE.md`.
3. Read `docs/SECURITY.md`.
4. Read `docs/HANDOFF.md`.
5. Read `docs/REFERENCES.md`.
6. Read `docs/PROJECT_MEMORY.md` if present.
7. Check branch, git status, latest commit.
8. Identify the next smallest action.
9. Do not trust stale handoff without checking repository state.
