# Worktree isolation examples

Worktrees are optional for parallel work.

Use these examples with `recipes/worktree-isolation.md`.

These examples are not copied into scaffolded projects by default.

## Files

- `parallel-session-handoff.md`: handoff template for isolated work.
- `claude-worktree-policy.md`: Claude-oriented policy example.
- `filled-parallel-session-handoff-risky-refactor.md`: filled example with merge and cleanup plan.
- `cleanup-checklist.md`: cleanup checklist after merge or rejection.
