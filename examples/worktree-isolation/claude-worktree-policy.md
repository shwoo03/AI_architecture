# Claude worktree policy example

## Policy

- Use worktrees for parallel edits.
- Do not copy secrets by default.
- Record branch and worktree names in handoff.
- Install dependencies explicitly per worktree.
- Merge or reject changes before cleanup.
- Clean up worktrees after merge/reject.
