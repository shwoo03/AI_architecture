# Research Rules

- Reference cards and proposals must be source-backed.
- Prefer local `AI_architecture_references` and approved cloned repos before network search.
- Network search or clone requires explicit user confirmation.
- Named references in the user goal take priority over unreviewed but unrelated repos.
- Record reuse mode as dependency, wrapper, partial copy, concept-only, or reject.
- Do not add dependencies or copy code until the review queue decision allows it.
